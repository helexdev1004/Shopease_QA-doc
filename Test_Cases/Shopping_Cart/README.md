# Shopping Cart Module

This folder contains QA documentation for the Shopping Cart functionality of the ShopEase e-commerce application.

## Scope

- Add Product to Cart
- Remove Product
- Update Quantity
- Cart Price Calculation
- Empty Cart
- Save Cart
- Session Persistence
- Guest User Cart
- Logged-in User Cart

## Documents

- Shopping Cart Test Cases
- Shopping Cart Bug Reports
- Shopping Cart Test Data
- Shopping Cart Execution Report
- Shopping Cart Traceability Matrix
