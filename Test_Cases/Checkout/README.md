# Checkout Module

This folder contains QA documentation for the Checkout functionality of the ShopEase e-commerce application.

## Scope

- Guest Checkout
- Registered User Checkout
- Shipping Information Validation
- Billing Information Validation
- Payment Method Selection
- Promo Code Validation
- Order Summary Verification
- Order Confirmation
- Error Handling

## Documents

- Checkout Test Cases
- Checkout Bug Reports
- Checkout Test Data
- Checkout Execution Report
- Checkout Traceability Matrix
