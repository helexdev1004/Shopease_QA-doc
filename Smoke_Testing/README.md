# Smoke Testing

This folder contains smoke testing artifacts for the ShopEase e-commerce application.

## Scope

- Build Verification
- Critical User Flows
- Login
- Product Search
- Shopping Cart
- Checkout
- Order Placement

## Documents

- Smoke Test Cases
- Smoke Execution Report

## Purpose

Smoke testing verifies that the most critical application functionalities work correctly before detailed functional and regression testing begins.
